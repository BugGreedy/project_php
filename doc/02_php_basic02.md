### 2-1_IF文による条件分岐
IF文の基本構成は下記のとおり。
```php
<?php
$n = rand(1,100);
  if($n % 2 == 0){ // ＄nが奇数のとき
  // 条件が成り立つとき
  echo $n.'は偶数';
  // 条件が成り立たないとき
  }else{  
    echo $n.'は奇数';
  }
}
?>
```
注意事項として、"{}"の付け忘れが挙げられる。</br>
`else`の前後や最後のとじかっこを忘れない。</br>
