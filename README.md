## はじめに
phpはサーバーのドキュメントルート配下にphpを置き、ブラウザから`http://~`の形式でアクセスしないと正しく解釈されない。(記述したソースがそのまま表示されるだけ)</br>
直接ブラウザに突っ込んでもソースのみの表示になる。</br>
MAMPなどのディストリビューションの場合はhtdocsの中にディレクトリを置く。</br>
もちろんapacheやDockerなど仮想サーバーが起動していないと表示されない。</br>
